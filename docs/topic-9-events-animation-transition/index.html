JavaScript Events

There are many JavaScript events that can be listened for. We will only be covering a few of the most common ones here, but for a more extensive list, you can visit: W3 Schools DOM Events

Events extend the Event Object and have a defined set of properties and methods that can be used. More information on the Event Object can be found here: W3 Schools Event Object
